# Toolasha Feature List

Complete list of all Toolasha features. For detailed documentation, see [DOCUMENTATION.md](DOCUMENTATION.md).

## Action Panel

- **Total Action Time** - Queue completion time displayed top-left
- **Action Panel Time Display** - Time/XP/level goals on action panels
- **Quick Input Buttons** - Preset buttons for 1/10/100/Max hours or counts
- **Foraging Profit** - Profit calculations for multi-outcome gathering actions
- **Action Queue Time** - Total time shown in queue menu
- **Output Totals** - Expected outputs calculated from input quantity
- **Max Produceable** - Shows how many items you can craft with current materials
- **Required Materials** - Shows total required and missing materials for crafting
- **Gathering Stats** - Profit and XP per hour on gathering actions
- **Hide Negative Profit** - Option to hide actions with negative profit
- **Sort by Profit** - Sort actions by profit per hour (descending)

## Tooltips

- **Market Prices** - 24-hour average ask/bid prices on item tooltips
- **Production Profit** - Crafting cost and profit calculations
- **Detailed Profit Breakdown** - Material costs table with individual prices
- **Expected Value** - EV calculations for openable containers (chests, bags, etc.)
- **Consumable Stats** - HP/MP restore rates and efficiency metrics
- **Enhancement Costs** - Cost calculations for enhancement upgrade paths
- **Gathering Sources** - Shows which gathering actions produce the item
- **Rare Drop Display** - Shows rare find drops from gathering actions

## Enhancement

- **Enhancement Tracker** - Track enhancement attempts, costs, and success rates per session
- **Enhancement Simulator** - Calculate optimal enhancement strategies and expected costs
- **Enhancement Path Display** - Show material and coin costs for upgrade paths
- **Consumed Items Detail** - Detailed breakdown for Philosopher's Mirror enhancement costs

## Economy & Inventory

- **Net Worth Display** - Total asset value displayed top-right of screen
- **Inventory Summary** - Detailed worth breakdown shown below inventory grid
- **Inventory Sorting** - Sort items by value, type, or other criteria
- **Stack Value Badges** - Display total value on item stack icons
- **Badge Price Display** - Show ask/bid prices on inventory items (independent of sorting)
- **Pricing Modes** - Choose Conservative/Hybrid/Optimistic for profit calculations
- **High Enhancement Cost** - Use calculated cost for +13 and above items
- **Cowbell Inclusion** - Option to include cowbells in net worth calculations

## Combat

- **Combat Score** - Gear score calculation displayed on profile panel
- **Zone Indices** - Zone index numbers shown on combat zone buttons
- **Ability Book Calculator** - Calculate books needed to reach target ability levels
- **Dungeon Tracker** - Track dungeon run times, wave progress, and team statistics
- **Dungeon Chat Annotations** - Colored timer annotations in party chat
- **Combat Simulator Integration** - Auto-import character and party data
- **Combat Summary** - Encounter statistics when returning from combat

## Tasks

- **Task Profit Display** - Total profit calculations for task rewards
- **Task Reroll Tracker** - Track reroll costs and spending over time
- **Task Zone Index** - Zone numbers displayed on task combat requirements
- **Task Sorter** - Sort tasks by skill type
- **Task Icons** - Visual icons displayed on task cards
- **Task Icons Dungeons** - Shows dungeon icons for combat tasks

## Market

- **Item Count Display** - Show inventory count on market item tiles
- **Listing Price Display** - Show per-unit and total prices on your listings
- **Market Listing Age** - Shows listing creation time
- **Market Filter** - Filter items by level, class, or equipment slot
- **Auto-Fill Market Price** - Automatically fill optimal market listing prices

## UI Enhancements

- **Skill XP Percentage** - Display percentage progress to next skill level
- **Skill Remaining XP** - Shows XP needed to reach next level
- **Equipment Level Display** - Show enhancement level on equipment icons
- **Key Zone Index** - Display zone number on dungeon key icons
- **Alchemy Item Dimming** - Dim unavailable alchemy ingredients
- **K/M/B Number Formatting** - Display large numbers as 1.5M instead of 1,500,000

## House

- **House Upgrade Costs** - Display upgrade costs with inventory availability comparison

## Notifications

- **Empty Queue Alert** - Browser notification when action queue completes
- **Network Alert** - Alert when market data fetch fails

## Settings & Configuration

- **Comprehensive Settings Panel** - Organized settings for all features
- **Enhancement Simulator Config** - Configure skill levels, tea buffs, and tool bonuses
- **Pricing Mode Selection** - Choose Conservative/Hybrid/Optimistic calculation methods
- **Expected Value Options** - Configure how container drops are displayed
- **Inventory Badge Options** - Customize badge display and price types
- **Dungeon Tracker Options** - UI visibility and chat annotation settings
- **Color Customization** - Extensive color customization for UI elements (15+ color options)
